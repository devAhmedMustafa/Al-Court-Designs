### 🧠 Story ID: US-008

**Feature**: Place order

### Actors 🧑‍💻

- Customer

***As a*** Customer,
***I want to*** place order of items in the cart,
 ***So that*** the restaurant got notified of my request.

### 🗃️ Data Dictionary

- Order number
- Customer Id
- Items
- Branch id
- Date
- Indoor/Takeaway
- Payment method
- Status
- Payment Status

### ✔️ Acceptance Criteria

- Customer is authenticated
- Restaurant is open
- Number of items is more than zero
